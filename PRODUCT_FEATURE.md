# AI Agent Academy — เอกสารฟีเจอร์ผลิตภัณฑ์ (สำหรับสร้างสื่อโปรโมท)

> เอกสารนี้รวบรวมจุดขาย ฟีเจอร์ และประโยชน์ของแพลตฟอร์ม เพื่อให้ทีมการตลาด/ครีเอเตอร์
> นำไปทำสื่อโปรโมท (โพสต์โซเชียล, แลนดิ้งเพจ, สคริปต์โฆษณา, สไลด์ขายของ) ได้ทันที
> ทุกฟีเจอร์ในเอกสารนี้ **มีอยู่จริงในระบบ** — หยิบไปใช้ได้โดยไม่ต้องกลัวเคลมเกินจริง

## 1. ภาพรวมผลิตภัณฑ์ (Product Snapshot)

**AI Agent Academy** คือเว็บระบบจัดการ "ร้าน/สถาบันสอนคอร์ส AI" แบบครบวงจร
ตั้งแต่ลูกค้าเลือกคอร์ส → จ่ายเงิน → แนบสลิป → ระบบตรวจสลิปและอนุมัติให้อัตโนมัติ
พร้อมหลังบ้านสำหรับแอดมินจัดการคอร์ส รอบเรียน และยอดสมัครในที่เดียว

- **กลุ่มผู้ใช้:** สถาบัน/ติวเตอร์/ครีเอเตอร์ที่ขายคอร์ส AI หรือคอร์สออนไลน์ในไทย
- **จุดต่าง:** ออกแบบมาเพื่อ "การจ่ายเงินแบบไทย" (โอนธนาคาร + พร้อมเพย์ + แนบสลิป) โดยเฉพาะ
- **คุณค่าหลัก:** ปิดการขายเร็วขึ้น ลดงานแอดมินตรวจสลิปด้วยมือ และไม่พลาดทุกการแจ้งโอน

### Tagline ที่หยิบไปใช้ได้
- "เปิดคอร์ส AI ของคุณ — ขาย รับเงิน ยืนยันที่นั่ง จบในระบบเดียว"
- "ลูกค้าแนบสลิป ระบบอนุมัติให้เอง ภายในไม่กี่วินาที"
- "ตรวจสลิปอัตโนมัติ พร้อมเตือนเข้า LINE ทุกการโอน"

## 2. จุดขายเด่นที่สุด (Hero Features)

ฟีเจอร์ 3 ตัวนี้คือพระเอกของสื่อโปรโมท ควรดันเป็นหัวข้อหลัก

### ⭐ 1) ตรวจสลิปอัตโนมัติ + อนุมัติทันที (Auto Slip Verification)
ลูกค้าแนบสลิปเสร็จ ระบบเชื่อม API ตรวจสลิปจริง แล้ว **ตรวจ 3 ชั้นให้อัตโนมัติ**:
ยอดเงินตรงกับค่าคอร์ส · บัญชีผู้รับถูกต้อง · ไม่ใช่สลิปที่เคยใช้แล้ว

- ผ่านทั้งหมด → **ยืนยันที่นั่งให้ทันที** ลูกค้าได้เริ่มเรียนเลย ไม่ต้องรอ
- ยังไม่เปิดใช้ระบบตรวจ → สลับเป็นโหมดแอดมินตรวจมือได้ (ยืดหยุ่น 100%)

> **มุมโปรโมท:** "ลูกค้าไม่ต้องรอแอดมินว่าง — จ่ายปุ๊บ เข้าเรียนปั๊บ"

### ⭐ 2) แจ้งเตือนเข้า LINE ทุกการแจ้งโอน (LINE Notifications)
ทุกครั้งที่มีลูกค้าแนบสลิป ระบบส่งข้อความสรุปเข้า **LINE Official Account** ของร้าน
(ชื่อคอร์ส, ชื่อลูกค้า, ยอดเงิน, ผลการตรวจ, ลิงก์ไปตรวจสอบ)

> **มุมโปรโมท:** "ไม่พลาดทุกออเดอร์ แม้ไม่ได้นั่งเฝ้าหน้าจอ"

### ⭐ 3) กันสลิปซ้ำ + กันที่นั่งเกิน (No Duplicate, No Overbooking)
- **กันสลิปปลอม/สลิปซ้ำ:** สลิปเดียวใช้ได้ครั้งเดียว ป้องกันการเอาสลิปเก่ามาใช้ซ้ำ
- **กันที่นั่งเต็มเกิน:** ตรวจจำนวนที่นั่งตอนยืนยันแบบ real-time ไม่มีรับเกินโควต้า

> **มุมโปรโมท:** "ปลอดภัยเรื่องเงิน ชัวร์เรื่องที่นั่ง"

## 3. ฟีเจอร์ฝั่งลูกค้า (Customer Experience)

| ฟีเจอร์ | ประโยชน์ที่ลูกค้าได้ |
|---|---|
| เลือกดูคอร์สจากแคตตาล็อก | เห็นคอร์สทั้งหมด ราคา และรายละเอียดชัดเจน |
| คอร์ส 2 แบบ: มีรอบเรียน / เรียนได้ทันที | เลือกได้ตามไลฟ์สไตล์ — จองรอบสด หรือเริ่มเรียนเลย |
| เห็นที่นั่งคงเหลือของแต่ละรอบ | ตัดสินใจง่าย รู้ว่ารอบไหนใกล้เต็ม |
| จ่ายด้วยโอนธนาคาร + พร้อมเพย์ QR | สแกนจ่ายได้ทันที QR สร้างตามยอดคอร์สอัตโนมัติ |
| แนบสลิปง่าย + บีบอัดรูปอัตโนมัติ | อัปโหลดไว ประหยัดเน็ต ไม่ต้องย่อรูปเอง |
| ติดตามสถานะการสมัคร | รู้ทุกขั้น: รอชำระ → ส่งสลิปแล้ว → ยืนยันแล้ว |
| สลิปไม่ผ่าน แจ้งเหตุผล + แนบใหม่ได้ทันที | ไม่ต้องทักแชทถามแอดมิน แก้เองได้เลย |
| ดูประวัติคอร์สที่สมัครทั้งหมด | จัดการคอร์สของตัวเองได้ในหน้าเดียว |

> **มุมโปรโมท:** "สมัครเรียนจบใน 3 ขั้น — เลือกคอร์ส · สแกนจ่าย · แนบสลิป"

## 4. ฟีเจอร์ฝั่งแอดมิน/เจ้าของร้าน (Back Office)

| ฟีเจอร์ | ประโยชน์ที่เจ้าของร้านได้ |
|---|---|
| แดชบอร์ดสรุป | เห็นยอดที่รอตรวจและภาพรวมรายได้ในที่เดียว |
| สร้าง/แก้ไขคอร์ส + เปิด-ปิดการขาย | จัดการแคตตาล็อกเองได้ ไม่ต้องพึ่งโปรแกรมเมอร์ |
| จัดการรอบเรียน + กำหนดจำนวนที่นั่ง | คุมโควต้าแต่ละรอบได้ละเอียด |
| ตรวจสลิป อนุมัติ/ปฏิเสธพร้อมเหตุผล | จัดการเคสที่ระบบส่งต่อให้ได้รวดเร็ว |
| เห็นผลตรวจสลิปอัตโนมัติประกอบการตัดสินใจ | รู้ทันทีว่าสลิปไหน "ผ่าน/ไม่ผ่าน/ต้องตรวจมือ" |
| ระบบสิทธิ์แยกแอดมิน/ลูกค้า | ปลอดภัย ข้อมูลหลังบ้านเข้าถึงได้เฉพาะแอดมิน |

> **มุมโปรโมท:** "เปิดร้านคอร์สได้เอง ทั้งหน้าร้านและหลังบ้าน ไม่ต้องจ้างทีมไอที"

## 5. ความน่าเชื่อถือ/ความปลอดภัย (Trust & Tech)

หยิบไปใช้ในสื่อที่ต้องการสร้างความมั่นใจ (B2B / นักลงทุน / ลูกค้าองค์กร)

- 🔒 **เก็บสลิปแบบ private** เปิดดูได้เฉพาะผ่านระบบที่ตรวจสิทธิ์แล้ว ไม่หลุดเป็นลิงก์สาธารณะ
- 🔒 **ตรวจสลิปกับ API จริง** ไม่ได้เดาจากรูป — ลดความเสี่ยงสลิปปลอม
- ⚙️ **ยืนยันที่นั่งแบบ transaction** กันสองคนแย่งที่นั่งสุดท้ายพร้อมกัน
- ☁️ **รองรับสเกลด้วย cloud storage** (Cloudflare R2) พร้อมโตตามจำนวนลูกค้า
- 🧩 **เลือก provider ตรวจสลิปได้** (เช่น EasySlip) หรือปิดไว้ใช้ตรวจมือก่อนก็ได้
- 🇹🇭 **ออกแบบเพื่อตลาดไทย** ภาษาไทยทั้งระบบ + ช่องทางจ่ายเงินแบบไทย

## 8. โครงเรื่องสำหรับวิดีโอ/สไลด์เดโม (Demo Flow)

ใช้เป็นสตอรีบอร์ดถ่ายคลิปสาธิตหรือทำสไลด์

1. **ฉากลูกค้า** — เลือกคอร์ส AI → เห็นราคาและที่นั่งคงเหลือ → กด "สมัครเรียน"
2. **ฉากจ่ายเงิน** — สแกน QR พร้อมเพย์ที่สร้างตามยอด → แนบสลิป (รูปย่ออัตโนมัติ)
3. **ฉาก "ว้าว"** — ระบบตรวจสลิปแล้วขึ้น "ยืนยันแล้ว ✅" ทันที โดยไม่มีคนกด
4. **ฉากเจ้าของร้าน** — ข้อความเด้งเข้า LINE + แดชบอร์ดอัปเดตยอดสมัครแบบ real-time
5. **ปิดท้าย** — โลโก้ + tagline + CTA
